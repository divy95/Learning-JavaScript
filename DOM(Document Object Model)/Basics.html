Manipulating ul with the help of DOM

const list=document.querySelector('ul') This will select the ul tag
const green=list.querySelector('li') this will select the 1st li
 green.style.backgroundColor='green'this will turn the first li background-color green
green.innerHTML=five & this will change the text one to five

QuerySelectorAll
const val=document.querySelectorAll('li')this will select all the li as an array
val[0].style.color='green' this will turn the first li color content red
